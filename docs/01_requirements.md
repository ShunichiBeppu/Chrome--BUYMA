# BUYMA 出品自動化 Chrome 拡張 要件定義

## 1. 目的
Googleスプレッドシートの一覧からBUYMA出品を自動化し、送信前にプレビューで停止できるChrome拡張を社内専用で運用する。

## 2. 前提
- BUYMAはログイン済みのブラウザで利用する
- Googleスプレッドシートはブラウザで開いている状態
- 拡張機能をONにするとBUYMAの出品ページを自動で開く
- 画像は拡張側で扱わない（入力しない）
- 社内専用配布（Chrome Web Store公開なし）

## 3. 入力データ（スプレッドシート列）
- 商品URL
- 商品名
- 公式サイト
- 買付地
- 色の系統
- サイズ
- 出品金額
- ユーロ
- エン
- 保存方法（下書き / 保存）
- 出品ステータス（未着手 / 完了）

## 4. 対象フロー
1. 拡張機能をON
2. スプレッドシートの「未着手」行を上から順に最大20件取得
3. BUYMA出品ページを新規タブで開く
4. 項目入力（商品情報/価格/サイズ/色/説明など）
5. 送信前にプレビューで停止（人の確認）
6. 手動で送信または保存
7. スプレッドシート上の出品ステータスを「完了」に書き換え

## 5. 操作仕様
- ワンクリックで一括開始
- 1回の実行で最大20件
- ステータス更新はシート画面上のセルを書き換え

## 6. スコープ外
- ログイン自動化
- 画像入力の自動化
- BUYMA以外のプラットフォーム対応
- 在庫/価格の自動更新

## 7. 例外・注意事項
- BUYMA画面構成変更により自動入力が失敗する可能性あり
- 入力途中でユーザーが画面操作を行う場合、誤動作の可能性あり
- 20件実行中は他タブ操作を控える推奨
